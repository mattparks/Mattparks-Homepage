Flounder Enginemore_vert
Flounder Engineclose

The Flounder Engine is a 3D engine built on LWJGL with no other librarys. The engine supports 3D and 2D along with API's for physics, maths, music, sounds and more. The goal of the engine was to create a skeleton engine for creating games without worrying about creating a bunch of new code that the engine alreay has. It is currently on the final stages on development, this has been a year project for me and will be used in future games by me.

DarkNoteclose

DarkNote is a 2D platformer game that uses a Perlin generator to create infinite assortments of music and levels to explore. We use a difficulty level on players to determine how hard each level should be, as the player goes throughout each level they will pick up experience, lives, health, points and attacks. DarkNote was going to be featured on a TV game show but that has sence fallen though.
